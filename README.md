Trilha de Documentação
Repositório com a resolução da Lista de Exercícios de Fixação de cada módulo da Trilha de Documentação da UniSENAI, disciplina do curso de Análise e Desenvolvimento de Sistemas (ADS).
Cada módulo aborda um aspecto da documentação técnica de software, utilizando o MKDocs como ferramenta principal para criação, versionamento, automação e publicação de documentação.
📚 Módulos
Módulo	Tema
01 - Introdução	Conceitos fundamentais de documentação de software: tipos, importância e boas práticas.
02 - Visão Geral sobre MKDocs	O que é o MKDocs, instalação, configuração inicial e sintaxe básica de Markdown.
03 - Criação de Documentação com MKDocs	Estrutura de projeto, temas, personalização visual e plugins.
04 - Versionamento e Deploy da Documentação	Integração com Git, publicação no GitHub Pages e deploy contínuo com GitHub Actions.
05 - Práticas de Escrita Técnica	Escrita clara e concisa, organização de conteúdo e formatação de documentação.
06 - Documentação Automatizada e API Docs	Documentação automatizada de código, especificação OpenAPI e integração com Swagger UI.
07 - Estudos de Caso e Aplicações Práticas	Aplicação do MKDocs em projetos reais, colaboração em equipe e documentação de APIs.
🛠️ Tecnologias e ferramentas abordadas
MKDocs + tema Material for MkDocs
Markdown
Git e GitHub
GitHub Pages e GitHub Actions (CI/CD)
OpenAPI e Swagger UI
