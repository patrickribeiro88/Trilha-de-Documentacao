# Trilha de Documentação - UniSENAI

Repositório com a resolução da **Lista de Exercícios de Fixação** de cada módulo da Trilha de Documentação da UniSENAI, disciplina do curso de Análise e Desenvolvimento de Sistemas (ADS).

Cada módulo aborda um aspecto da documentação técnica de software, utilizando o **MKDocs** como ferramenta principal para criação, versionamento, automação e publicação de documentação.

## 📚 Módulos

| Módulo | Tema |
|---|---|
| [01 - Introdução](./01%20-%20Introdução) | Conceitos fundamentais de documentação de software: tipos, importância e boas práticas. |
| [02 - Visão Geral sobre MKDocs](./02%20-%20Visão%20Geral%20sobre%20MKDocs) | O que é o MKDocs, instalação, configuração inicial e sintaxe básica de Markdown. |
| [03 - Criação de Documentação com MKDocs](./03%20-%20Criação%20de%20Documentação%20com%20MKDocs) | Estrutura de projeto, temas, personalização visual e plugins. |
| [04 - Versionamento e Deploy da Documentação](./04%20-%20Versionamento%20e%20Deploy%20da%20Documentação) | Integração com Git, publicação no GitHub Pages e deploy contínuo com GitHub Actions. |
| [05 - Práticas de Escrita Técnica](./05%20-%20Práticas%20de%20Escrita%20Técnica) | Escrita clara e concisa, organização de conteúdo e formatação de documentação. |
| [06 - Documentação Automatizada e API Docs](./06%20-%20Documentação%20Automatizada%20e%20API%20Docs) | Documentação automatizada de código, especificação OpenAPI e integração com Swagger UI. |
| [07 - Estudos de Caso e Aplicações Práticas](./07%20-%20Estudos%20de%20Caso%20e%20Aplicações%20Práticas) | Aplicação do MKDocs em projetos reais, colaboração em equipe e documentação de APIs. |

## 🛠️ Tecnologias e ferramentas abordadas

- [MKDocs](https://www.mkdocs.org/) + tema [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/)
- Markdown
- Git e GitHub
- GitHub Pages e GitHub Actions (CI/CD)
- OpenAPI e Swagger UI
